# Supermarket-sales-data-analysis (EDA)

## Project overview
This project focuses on performing exploratory data analysis(EDA) on supermarket sales dataset to uncover trends ,analyze profitibility and identify key drivers of growth.

## Key insights discovered
1.Sales seasonality:Documented a massive sales spike in Q4(late parts of the year),likely driven by holiday shopping and promotional events.
2.Profit margins:Discovered that high sales volume does not always mean high profit margins,highlighting the need for optimized pricing stratagies.
3.Hero products :Identified key products like 'Hon Practical Foundations 30 x 60 Training Table, Light Gray/Charcoal'  or 'Hewlett Packard 310 Color Digital Copier'  as the primary revenue generators.
